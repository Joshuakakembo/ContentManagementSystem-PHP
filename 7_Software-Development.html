Methodology

On a large scale Hostalite uses an agile software development methodology that is used to develop the systems. Execution of this methodology requires Hostalite and the client to work together across various phases to define and build features and functionality of the systems in an iterative and incremental manner.  This methodology adopts a feature based approach. As such iterations shall continue until all features have been built.

This methodology consists of the following phases:

  • Planning
  • Requirements Specification
  • Analysis and Design
  • Development and Implementation
  • Testing
  • Deployment and Evaluation
